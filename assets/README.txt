Place TJ logo files in this folder so the launcher can display branding.

Supported logo filenames:
- tj_logo.png (recommended)
- tj_logo.gif
- tj_logo.ppm
- tj_logo.pgm

Optional app icon filename:
- tj_icon.ico

Example:
1) Copy the company logo file here and rename it to `tj_logo.png`.
2) Copy an icon file here and rename it to `tj_icon.ico` (optional).
